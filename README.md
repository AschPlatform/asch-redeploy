# asch-redeploy
A hassle-free local asch environment

# Install
```
npm install -g asch-redeploy
```

# Usage
```
git clone https://github.com/aschplatform/asch-dapp-helloworld
cd asch-dapp-helloworld

# watch for changes
asch-redeploy .
```
